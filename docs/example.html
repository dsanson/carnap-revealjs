Carnap + Pandoc + Reveal.js

David Sanson

Goal and Methods

Embed Carnap exercises into
a reveal.js slideshow
generated by pandoc

Basic Idea

  • Write the slide show in markdown, following these instructions
  • Include Carnap exercises using one of the formats described below
  • Convert to a slide show using pandoc and a custom template, that adds links to the appropriate CSS and JS files

Examples

Derivation (raw html)

<pre data-carnap-type="proofchecker"
           data-carnap-goal=":|-:((P->Q)->P)->P"
           data-carnap-submission="none"
           data-carnap-system="prop"
           data-carnap-guides="montague"
           data-carnap-options="fonts">
Show ((P -> Q)-> P) -> P
</pre>
Show ((P -> Q)-> P) -> P
